# Contributing

Thanks for your interest in improving this guide!

## How to Contribute

### Reporting Issues
- Found an error or outdated info? [Open an issue](https://github.com/KoreanThinker/tailscale-ssh-tmux-claude-code/issues/new).
- Include your OS, tool versions, and steps to reproduce.

### Improving Documentation
1. Fork this repository
2. Create a branch: `git checkout -b improve-docs`
3. Make your changes
4. Submit a pull request

### Translation
- English docs: `docs/en/`
- Korean docs: `docs/ko/`
- Want to add another language? Create `docs/<lang-code>/` and translate all 7 guides.

## Writing Guidelines

- **Be specific**: Include actual commands, expected outputs, and common errors.
- **Be concise**: Respect the reader's time. No filler.
- **Test everything**: Every command in this guide should work when copy-pasted.
- **Keep it bilingual**: If you update an English doc, please update the Korean equivalent too (or note it in the PR).

## Code of Conduct

Be kind. Be helpful. That's it.
